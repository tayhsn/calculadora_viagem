# Calculadora de Viagem 



## Sobre

O programa recebe uma lista de despesas e uma lista de pessoas, é necessário fornecer ambas os dados, então ele calcula o gasto total das despesas, e divide pela quantidade em pessoas. Em seguida, ele retorna um mapa com o nome das pessoas e o valor à pagar.

## Como usar

- Clone o repositorio usando ```git clone https://github.com/tayhsn/calculadora_viagem```
- Abra a pasta no seu editor de codigo favorito
- Abra o terminal e digite ```npm install``` 

- Altere o nome das pessoas no arquivo ```src/mocks/pessoas.js```, seguindo o padrão do arquivo.
- Altere as despesas no arquivo ```src/mocks/despesas.js```, seguindo o padrão do arquivo.
- No terminal digite ```npm start```, o resultado deve agora aparecer pra você.
- A calculadora atualiza o resultado a cada modificação no código. Altere os arquivos pessoas e despesas a vontade!
